# RP2040 I2C

Este dispositivo configura a I2C da placa Raspberry Pi Pico RP2040.

## Configuração

Este dispositivo permite a você configurar os pinos da I2C pelo painel de inspeção, pelo menu, de forma simples.

```
  +----------------------------+
  |  SDA Pin   [   GP4 ▼] Pin  |
  |  SCL Pin   [   GP5 ▼] Pin  |
  |  Frequency [100000 ▼] bps  |
  +----------------------------+
```
