# RP2040 I2C

This device configures I2C from the Raspberry Pi Pico RP2040 board.

## Comfiguration

This component allows you to configure the I2C pins by the inspection panel, in a simple way.

```
  +----------------------------+
  |  SDA Pin   [   GP4 ▼] Pin  |
  |  SCL Pin   [   GP5 ▼] Pin  |
  |  Frequency [100000 ▼] bps  |
  +----------------------------+
```
